## What is AutomateMyPC.AI?

**AutomateMyPC.AI** is a free, AI-powered desktop automation tool built specifically for Windows. Instead of learning scripts, syntax, or programming languages, you simply describe a task in plain English — and the tool executes it on your PC instantly.

Most Windows automation tools — PowerShell, AutoHotkey, Task Scheduler — require technical knowledge to use effectively. AutomateMyPC.AI removes that barrier entirely. If you can type a sentence, you can automate your PC.

> *"If you can describe it, your PC can do it."*

## 🚀 Quick Start

### Requirements

- Windows 10 or 11 (64-bit)
- Intel i3 / AMD equivalent or better
- 4 GB RAM minimum (8 GB recommended)
- 200 MB free disk space
- .NET Framework 4.8 or later
- Internet connection (required for AI features)

### Installation

1. **Download** the installer from [automatemypc.ai/download](https://sites.google.com/iit.ac.lk/automate-my-pc/download)
2. **Run** `AutomateMyPC_Setup.exe` and click "Run" when prompted by Windows security
3. **Follow** the setup wizard (takes about 60 seconds)
4. **Launch** AutomateMyPC.AI and type your first instruction
